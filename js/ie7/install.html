Installation

Follow these simple instructions to get IE7 working immediately on your server:

  • download the latest IE7 ZIP file

  • extract the contents to a directory on your server

  • include the IE7 JavaScript library in the HTML page you wish to test

    <!-- compliance patch for microsoft browsers -->
    <!--[if lt IE 7]>
    <script src="/ie7/ie7-standard.js" type="text/javascript">
    </script>
    <![endif]-->

  • make sure this also points to the directory where IE7 was installed

  • open the page in your web browser

  • the page should now be IE7 enabled.

You may extract the contents of the ZIP file to your hard disk if you do not have access to a web server.

Troubleshooting

If things are not working as expected try doing the following:

  • append ?ie7_debug=true to the URL of your test page
  • refresh the page
  • you will either get a message informing you of a successful load, or you will get an error message
  • you can report errors on the IE7 Forum.

If you wish to debug IE7 yourself, link to the libraries in the src directory.
