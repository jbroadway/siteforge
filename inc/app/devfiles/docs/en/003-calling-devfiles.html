Calling DevFiles from a Template

To insert DevFiles into your web site, open your main XT templates (in the inc/html folder, usually called html.tpl) and add the following line:


<xt:box name="devfiles/index" />

Additionally, you can use the following optional attributes to further configure the way DevFiles displays:

Attribute Description
highlight="no" Turns off the alternating colour highlighting.
posting="private" Turns posting privileges off for users who can't write to the specified access level. You can use this to privide downloads to public or semi-public sites where users can't add their own files.
title="Related Downloads" Changes the displayed title from "DevFiles" to anything you want. Note that this string is passed through intl_get(), so that your title can appear in any language you want.
showFileInfo="no" Hides the second line of information (user name and date/time posted). Handy for public web sites.
hideIfEmpty="yes" Hides DevFiles altogether if there are no files for the current page. Note: DevFiles still displays for admin-level users, so you can still add files if you're an administrator and this setting is used.

An example of using these options might be:


<xt:box
	name="devfiles/index"
	highlight="no"
	posting="private"
	title="Related Downloads"
	showFileInfo="no"
	hideIfEmpty="yes"
/>
