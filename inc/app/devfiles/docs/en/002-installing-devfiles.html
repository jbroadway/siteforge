Installing DevFiles

From the Unix command-line, enter the following:


mysql -p -u USER DBNAME < inc/app/devfiles/install/install-mysql.sql

Then enter your password when prompted. You may also use an alternate MySQL client, such as phpMyAdmin, if you prefer. This will create the necessary database tables that DevFiles needs to store your file listings.

Next you need to set the permissions on the directory where files are stored. To do this, enter the following commands on the command-line:


bash$ cd inc/app/devfiles
bash$ chmod o+w data
