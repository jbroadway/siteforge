Calling DevFiles from a Template

In your main XT templates (in the inc/html folder) add the following line:


<xt:box name="devnotes/index" />

Additionally, you can use the following optional attributes to further configure the way DevNotes displays:

Attribute Description
highlight="no" Turns off the alternating colour highlighting.
posting="private" Turns posting privileges off for users who can't write to the specified access level. You can use this to privide downloads to public or semi-public sites where users can't add their own files.
title="Peanut Gallery" Changes the displayed title from "DevNotes" to anything you want. Note that this string is passed through intl_get(), so that your title can appear in any language you want.

An example of using these options might be:


<xt:box
	name="devfiles/index"
	highlight="no"
	posting="private"
	title="Peanut Gallery"
/>
