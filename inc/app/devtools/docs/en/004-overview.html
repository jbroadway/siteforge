Command Overview

devtools is always an ongoing project, and new tools are added all the time.  Currently, the following tools exist:

ToolDescription
access.sh
Creates a new access.php file.
app.sh
Creates a new app structure with all the necessary folders and default files.
box.sh
Creates a new box structure (folder) and default index.php, settings.php, and access.php files.
form.sh
Creates a new form structure (folder) and default index.php, settings.php, and access.php files.
tpl.sh
Creates a new template set structure (folder) and default html.default.tpl, site.css, config.ini.php, and pix subfolder.
