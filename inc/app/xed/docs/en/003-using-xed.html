Using Xed

To create a Xed widget in one of your forms, simply add the following elements to your form settings.php file:


[field_name]
type = xed.Widget.Xeditor

This will create the necessary widget. Now in the main [Form] section, add the following line:


extra = "onsubmit=`xed_copy_value (this, 'field_name')`"

This handles the task of copying the editor value from the iframe to a hidden variable within the form (created automatically for you) that will pass along the editor contents upon form submission.

You can also call the Xed MailForm widget directly from PHP code, if your form doesn’t happen to use a settings.php file for some reason. In this case you could simply say all at once:


$widget =& $form->addWidget ('xed.Widget.Xeditor', 'field_name');
$form->extra = 'onsubmit="xed_copy_value (this, \'field_name\')"';
