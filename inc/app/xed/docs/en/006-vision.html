Xed 2.0 Vision

Xed version 1.0 is already a very simple yet featureful HTML editing widget for the Sitellite CMS. It boasts features such as:

  1. Cross-browser/cross-platform support
  2. Complete editing functionality, including basic table creation
  3. Adding images from server or local hard-drive (via Image Chooser app)
  4. Implemented as a MailForm widget, so that it can be integrated into Sitellite apps as easily as possible

But we want to go further than that in the next version. We want to be the best browser-based WYSIWYG editor out there. And for this we need a new set of goals. So far, these include:

  1. Keep the UI and the number of features at a minimum
  2. Improved table editing (add/remove columns and rows, headers, table properties, etc.)
  3. Spell checker app (w/ custom dictionaries)
  4. Multiple instances per HTML page
  5. XML editing (using schemas to build the toolbar, but no XSLT —CSS can do the same job more easily)
  6. Box-maker and form-maker apps to be able to choose, configure, and insert boxes and forms dynamically into any page (using <xt-box name="..." /> syntax and some basic CSS)
  7. Improved I18n support (translated UI, UTF-8 support)
  8. Full-screen editing mode
  9. Content templates for improving consistency of content throughout sites
