Browser Requirements

Xed is a cross-platform in-browser WYSIWYG editor, but that doesn’t mean every browser is made equally now. Xed officially supports the following browsers on the specified platforms:

  1. Microsoft Internet Explorer 5.5 or greater on Windows only. MSIE on Mac is a very different product than its Windows counterpart, and as such doesn’t support the contentEditable API at all.
  2. Mozilla 1.3 or greater on any operating system. Tested OSes include Windows, Mac OS X, Linux, and FreeBSD.
